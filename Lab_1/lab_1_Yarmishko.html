Контрольні запитання

1. Що таке pip і conda та чим вони відрізняються між собою?

pip — це стандартний інструмент для встановлення бібліотек Python. Він дозволяє встановлювати пакети, але не керує віртуальними середовищами.

conda — це інструмент, який використовується для створення та управління віртуальними середовищами, а також для встановлення бібліотек.

Основна відмінність полягає в тому, що pip працює тільки з пакетами, а conda — і з пакетами, і з віртуальними середовищами.

2. Як буде виглядати виконання пунктів 1–3 індивідуального завдання за допомогою Virtualenv?

При використанні Virtualenv створюється та активується віртуальне середовище. Після цього всі необхідні бібліотеки встановлюються за допомогою pip.

На відміну від conda, Virtualenv не має власного менеджера пакетів і не дозволяє керувати версіями Python.

3. Як виконати візуалізацію зображення засобами matplotlib?

Для візуалізації зображення використовується бібліотека matplotlib. Зображення відображається за допомогою функції plt.imshow().

Для напівтонового зображення додатково вказується параметр cmap="gray".

4. Як виконати компіляцію документа Jupyter Notebook без використання командного рядка?

Компіляцію Notebook у формат HTML можна виконати безпосередньо з інтерфейсу Jupyter, обравши пункт меню File → Export Notebook As → HTML.
